1. Aim : Design a dynamic web app like E- Commerce product using HTML,
CSS & JS. Also use bootstrap CDN if required.

2. Objectives 
To understand the concept of dynamic web applications.
To learn the role of HTML in structuring a website.
To understand how CSS is used for styling and layout design.
To implement JavaScript for adding interactivity and responsiveness.
To understand Bootstrap CDN for developing a responsive and mobile-friendly interface.
To integrate multiple front-end technologies to build a complete web application.

3. Software Requirements:
Text Editor/IDE: Code editor such as Visual Studio Code, Sublime Text, or Atom
for writing HTML, CSS, and JavaScript
Web Browser: Modern web browsers like Google Chrome, Mozilla Firefox,
Safari, or Microsoft Edge with JavaScript enabled for testing the dynamic
functionality
Version Control (Optional): Git for code versioning and project management
Bootstrap CDN (Optional): If required, include Bootstrap CSS and JS via CDN
links for additional styling frameworks and components

4. Hardware Requirements:
Computer System: Standard desktop or laptop computer with adequate
processing power
Display: Monitor or screen capable of rendering web content at various
resolutions
Storage: Sufficient disk space for storing project files and images
Internet Connection: Required for loading external image resources and
optional Bootstrap CDN


5. Theory 
i) HTML for structure
HTML (HyperText Markup Language) is used to define the structure of the e-commerce website for selling clothes, shoes, and caps.
HTML provides:
Headings (<h1> to <h6> ) for titles like product categories
Paragraphs (<p>) for product descriptions
Images (<img>) to display products
Links (<a>) for navigation between pages
Product cards for clothes, shoes, and caps
Navigation bar for Home, Products, Cart, and Contact sections
Sections like Home, Categories, Product Listings, and Contact

ii) CSS for styling
CSS (Cascading Style Sheets) is used to style and design the e-commerce website.
CSS controls:
Colors for branding and product highlighting
Fonts for better readability
Backgrounds to enhance visual appeal
Spacing and alignment for neat layout
Hover effects on product cards and buttons
Layout design for organizing products in grids
CSS makes the website visually attractive and improves user experience.

iii) JavaScript for responsiveness
JavaScript is used to make the website interactive and responsive to user actions.
JavaScript helps in:
Adding products (clothes, shoes, caps) to the cart
Calculating total price dynamically
Showing alerts (e.g., item added to cart)
Updating cart items and quantities dynamically

iv) Bootstrap CDN
Bootstrap is a front-end framework used to build responsive and mobile-first e-commerce websites.
Bootstrap provides:
Grid system for arranging product cards
Responsive layout for mobile and desktop views
Pre-designed buttons for actions like “Add to Cart”
Cards for displaying products
Navigation bars for easy site navigation
Forms for contact or user input
