5. Theory
i) Interactive Data Visualization Dashboard
A data visualization dashboard is a graphical interface that displays data using charts, graphs,
and visual indicators. It helps users understand patterns, trends, and insights easily.
In a Weather Dashboard, data such as:
● Temperature
● Humidity
● Wind speed
● Rainfall
● Weekly forecast

can be displayed using bar charts, line graphs, or pie charts.
Interactive dashboards allow users to:
● Hover over charts to see exact values
● Filter data by date
● View different types of graphs
● Update data dynamically

ii) Chart.js
Chart.js is a simple and lightweight JavaScript library used for creating responsive and
interactive charts.
Features of Chart.js:
● Easy to use
● Supports bar, line, pie, doughnut, radar charts
● Responsive design
● Animated charts
● Tooltip support
Chart.js works using the &lt;canvas&gt; element in HTML.
Example of Including Chart.js via CDN:

&lt;script src=&quot;https://cdn.jsdelivr.net/npm/chart.js&quot;&gt;&lt;/script&gt;

Chart.js is ideal for beginners and small to medium dashboards.

iii) D3.js
D3.js (Data-Driven Documents) is a powerful JavaScript library used for creating complex and
customized data visualizations.
Features of D3.js:
● Highly customizable
● Works with SVG, HTML, and CSS
● Suitable for complex data visualization
● Supports animations and transitions
● Data binding capability
D3.js provides more control than Chart.js but is more complex.
Example of Including D3.js via CDN:

&lt;script src=&quot;https://d3js.org/d3.v7.min.js&quot;&gt;&lt;/script&gt;

Unlike Chart.js, D3.js does not provide ready-made charts. Instead, developers manually define
shapes, axes, and scales using JavaScript.
D3.js is suitable for:
● Advanced dashboards
● Real-time data visualization
● Custom charts and infographics
