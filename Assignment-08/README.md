DirectChat is a real-time messaging platform developed using the MERN stack (MongoDB, Express,
React, Node.js) and enhanced with Socket.io for bidirectional communication. The application enables
users to engage in instant one-on-one and group conversations without delays caused by traditional
request-response mechanisms. By utilizing WebSocket-based communication, the system ensures
real-time message delivery, typing indicators, read receipts, and live presence tracking.
The platform also supports media sharing, profile customization, and contact discovery, making it a
comprehensive communication solution. All data, including messages and user information, is stored
persistently in MongoDB. The system is designed with a responsive user interface and supports both
light and dark themes, ensuring a modern user experience across devices.
The application follows an event-driven architecture using WebSockets, allowing instant server-to-client
communication without repeated HTTP requests. Users can securely register, log in, manage profiles,
and interact with other users through a responsive interface.
This project demonstrates the practical implementation of real-time systems and showcases how modern
web technologies can be combined to build scalable, interactive communication platforms.
